# What Makes a Country Happy? - Data Storytelling App

## Project Overview

This project is a Data Storytelling Application developed using Python, Pandas, NumPy, Plotly, and Streamlit.

The application explores the World Happiness Report 2023 dataset and presents a story about how different factors such as GDP, social support, health, and freedom influence happiness across countries.

The goal of this project is to analyze the dataset, discover meaningful patterns, and communicate insights through visualizations and storytelling.

## Objectives

* Understand the factors affecting happiness.
* Perform Exploratory Data Analysis (EDA).
* Analyze relationships between different variables.
* Present insights through visualizations.
* Provide meaningful findings and recommendations.

## Dataset Information

**Dataset Name:** World Happiness Report 2023

**Dataset File:** WHR2023.csv

The dataset contains information about:

* Happiness Score
* GDP per Capita
* Social Support
* Healthy Life Expectancy
* Freedom to Make Life Choices
* Generosity
* Perceptions of Corruption

## Installation

Install the required libraries using:

```bash
pip install -r requirements.txt
```

## How to Run the Project

Open Command Prompt and navigate to the project folder.

Run:

```bash
python -m streamlit run data_storytelling_app.py
```

After execution, Streamlit will generate a local URL. Open the URL in your browser to access the application.

## Key Findings

1. Countries with higher GDP generally report higher happiness scores.

2. Social support contributes positively to overall well-being.

3. Healthy life expectancy is associated with happiness.

4. Freedom to make life choices influences life satisfaction.

5. Happiness is affected by multiple economic and social factors.

## Conclusion

The analysis shows that happiness is influenced by a combination of economic, social, and health-related factors. Countries with stronger economies, better social support, improved health conditions, and greater freedom generally report higher happiness scores.
